## I. Giới thiệu dự án
Dự án này là báo cáo thực hành về việc sử dụng AI có trách nhiệm, được trình bày dưới dạng Landing Page hiện đại. Nội dung tập trung vào phân tích chính sách, đạo đức và xây dựng bộ nguyên tắc cá nhân trong môi trường đại học.

## II. Công nghệ sử dụng
- **Frontend:** React.js / Next.js (hoặc HTML/Tailwind CSS tùy theo code m xuất ra).
- **Hiệu ứng:** Framer Motion (Scroll reveal), Three.js (3D background), Glassmorphism UI.
- **Deployment:** GitHub Pages.
- **Domain:** Custom domain .id.vn.

## III. Tóm tắt nội dung báo cáo
### 1. Phân tích chính sách
Nghiên cứu dựa trên khung tham chiếu của **Viện Công nghệ Massachusetts (MIT)** về trách nhiệm tuyệt đối và bảo vệ năng lực cốt lõi, đối chiếu với thực trạng tại VNU.

### 2. Case Study: Hệ thống Judge Online (C++)
- Sử dụng AI để lên ý tưởng Class Diagram và Debug lỗi memory leak.
- Tinh chỉnh bằng kỹ thuật cá nhân (Smart Pointers) và verify bằng Valgrind.

### 3. Đạo đức và Nguyên tắc cá nhân
- Phân tích ranh giới giữa hỗ trợ và gian lận học thuật.
- Xây dựng 5 nguyên tắc vàng: **Zero-Trust, Think First, Privacy, Transparency, Fundamentals**.

## IV. Hướng dẫn chạy dự án cục bộ
1. Clone repository: `git clone [link-repo]`
2. Cài đặt dependency: `npm install`
3. Chạy dev server: `npm run dev`
